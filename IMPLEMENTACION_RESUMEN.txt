Tabla: header_footer

CREATE TABLE `header_footer` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `tipo` enum('header','footer') NOT NULL UNIQUE,
  `titulo` varchar(200) DEFAULT NULL,
  `contenido` longtext NOT NULL,
  `estado` enum('activo','inactivo') DEFAULT 'activo',
  `fecha_creacion` timestamp NOT NULL DEFAULT current_timestamp(),
  `fecha_actualizacion` timestamp NOT NULL DEFAULT current_timestamp() ON UPDATE current_timestamp(),
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_general_ci;

P: ¿Dónde se guarda el header y footer?
R: En la tabla `header_footer` de la base de datos

P: ¿Se ve en todas las páginas?
R: Solo si integras mostrar_header_cache() en cada página

P: ¿Puedo usar código dinámico (PHP)?
R: No directamente en el editor, pero puedes hacerlo programáticamente

P: ¿Se puede desactivar?
R: Sí, cambia el estado a "Inactivo"

P: ¿Los cambios son inmediatos?
R: Sí, con caché se actualiza automáticamente

P: ¿Puedo tener múltiples headers?
R: Solo uno activo a la vez (tabla con clave única)

P: ¿Cómo revierté los cambios?
R: Usa el botón "Recargar" para descartar

P: ¿Se puede exportar?
R: Sí, desde phpMyAdmin puedes exportar la tabla
