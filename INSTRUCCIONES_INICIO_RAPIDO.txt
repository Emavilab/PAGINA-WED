## ⚡ 3 Pasos para Empezar

### PASO 1️⃣: Crear la tabla en la BD (30 segundos)

**Opción A - Más fácil (Recomendado):**
```
1. Abre tu navegador
2. Ve a: http://localhost/PAGINA-WED/setup_header_footer.php
3. Verás un mensaje confirmando la creación
4. ¡Listo!
```

**Opción B - Por terminal:**
```bash
# En la carpeta del proyecto
mysql -u root negocio_web < database/agregar_header_footer.sql
```

---

### PASO 2️⃣: Acceder al Panel de Control (10 segundos)

```
1. Abre http://localhost/PAGINA-WED/admin/configuracion.php
2. Busca el tab "Header y Footer" (última pestaña)
3. Haz clic en él
```

**Screenshot:**
```
[Marcas] [Métodos de Envío] [Métodos de Pago] [Config General] [Banners] [Hero] [Header y Footer] ← AQUÍ
```

---

### PASO 3️⃣: Editar Header y Footer (2 minutos)

**Para el Header:**
```
1. En la sección izquierda, escribe el título en "Título"
2. En "Código HTML del Header", pega tu HTML
3. Selecciona estado "Activo"
4. Haz clic en "Guardar Header"
5. Verifica en la previsualización
```

**Para el Footer:**
- El proceso es idéntico en la sección derecha

## 🔗 Integración en tus Páginas

Una vez guardado el header y footer, úsalos en tus archivos:

**En tu `index.php`:**
```php
<?php
require_once 'core/header_footer_helper.php';
?>
<!DOCTYPE html>
<html>
<head>...</head>
<body>
    <?php mostrar_header_cache(); ?>
    
    <!-- Tu contenido aquí -->
    
    <?php mostrar_footer_cache(); ?>
</body>
</html>
```

**Copiar a estos archivos:**
- `index.php`
- `client/categoria.php`
- `client/carrito.php`
- `pages/contactanos.php`
- Y los demás archivos de tu sitio

---

## ✨ Funciones Disponibles

```php
// Opción 1: Sin caché (una query por cada llamada)
mostrar_header();
mostrar_footer();

// Opción 2: Con caché (RECOMENDADO - mejor performance)
mostrar_header_cache();
mostrar_footer_cache();

// Opción 3: Solo obtener el HTML sin mostrar
$header_html = obtener_header_cache();
$footer_html = obtener_footer_cache();
```

## ❓ Preguntas Frecuentes

**P: ¿Dónde guarda los cambios?**  
R: En la tabla `header_footer` de la base de datos

**P: ¿Se puede ver en todas las páginas?**  
R: Sí, si integras `mostrar_header_cache()` y `mostrar_footer_cache()` en cada página

**P: ¿Se puede desactivar?**  
R: Sí, cambia el estado a "Inactivo" en el panel

**P: ¿Se puede revertir?**  
R: Sí, tienes el botón "Recargar" para descartar cambios no guardados

**P: ¿Puedo usar código dinámico (PHP)?**  
R: No directly, pero puedes crear un archivo intermedio que lo procese

## 🚨 Si Algo va Mal

### El tab no aparece:
```
✓ Asegúrate de estar logueado como ADMIN
✓ Recarga la página (Ctrl+F5)
✓ Verifica que el archivo api_header_footer.php existe
```

### No se guarda:
```
✓ Comprueba que el contenido no esté vacío
✓ Abre DevTools (F12) y revisa la consola
✓ Verifica que la tabla se creó: SELECT * FROM header_footer;
```

### No aparece en la página:
```
✓ Integra mostrar_header_cache() en tu HTML
✓ Verifica que el estado sea "Activo"
✓ Recarga la página sin caché (Ctrl+F5)
```
