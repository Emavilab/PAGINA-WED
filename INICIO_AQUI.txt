⚡ 4 PASOS PARA EMPEZAR

  1️⃣  CREAR TABLA EN BD (1 minuto)
      
      Opción A - Automático:
      └─ Visita: http://localhost/PAGINA-WED/setup_header_footer.php
      
      Opción B - Manual:
      └─ Ejecuta script SQL: database/agregar_header_footer.sql

  
  2️⃣  ACCEDER AL PANEL (30 segundos)
      
      └─ Abre: http://localhost/PAGINA-WED/admin/configuracion.php
      └─ Busca el tab "Header y Footer" (última opción)
      └─ ¡Haz clic!

  
  3️⃣  EDITAR CONTENIDO (2 minutos)
      
      Header:
      ├─ Escribe: HTML de tu encabezado
      ├─ Selecciona: Estado (Activo)
      └─ Haz clic: "Guardar Header"
      
      Footer:
      ├─ Escribe: HTML de tu pie de página
      ├─ Selecciona: Estado (Activo)
      └─ Haz clic: "Guardar Footer"

  
  4️⃣  INTEGRAR EN TUS PÁGINAS
      
      En cada archivo que necesite header/footer:
      
      // En el inicio del archivo:
      <?php require_once 'core/header_footer_helper.php'; ?>
      
      // Dentro del <body>, al inicio:
      <?php mostrar_header_cache(); ?>
      
      // Dentro del <body>, al final:
      <?php mostrar_footer_cache(); ?>

🚀 FUNCIONES DISPONIBLES

Importa en tus archivos:
  <?php require_once 'core/header_footer_helper.php'; ?>

Luego puedes usar:

  1. obtener_header()
     Retorna HTML del header como string

  2. mostrar_header()
     Imprime HTML del header

  3. obtener_header_cache()
     Retorna HTML con caché (más rápido)

  4. mostrar_header_cache()
     Imprime HTML con caché ⭐ RECOMENDADO

  5. obtener_footer()
  6. mostrar_footer()
  7. obtener_footer_cache()
  8. mostrar_footer_cache() ⭐ RECOMENDADO

💡 EJEMPLO RÁPIDO

<?php
// 1. Requerir helpers
require_once 'core/header_footer_helper.php';
?>
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Mi Tienda</title>
</head>
<body>
    
    <!-- 2. Mostrar header -->
    <?php mostrar_header_cache(); ?>
    
    <!-- 3. Tu contenido aquí -->
    <main>
        <h1>Bienvenido</h1>
        <p>Tu contenido...</p>
    </main>
    
    <!-- 4. Mostrar footer -->
    <?php mostrar_footer_cache(); ?>
    
</body>
</html>

Tabla: header_footer

Estructura:
┌─────────────┬──────────────┬──────────────────┐
│ Campo       │ Tipo         │ Descripción      │
├─────────────┼──────────────┼──────────────────┤
│ id          │ int(11)      │ ID único         │
│ tipo        │ enum         │ 'header'/'footer'│
│ titulo      │ varchar(200) │ Título opcional  │
│ contenido   │ longtext     │ HTML completo    │
│ estado      │ enum         │ 'activo'         │
│ fecha_crea. │ timestamp    │ Fecha creación   │
